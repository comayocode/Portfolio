Hola! soy Ronald, me encanta el desarrollo de software, diseño y creación de páginas web. Mi interés por el desarrollo comenzó en el 2017 cuando llevaron robots al colegio, los armamos y nos enseñaron a programar sus funciones para caminar.

Landing Page Comayan Service

Video de demostración:
Un poco sobre el proyecto

Este fue mi primer proyecto o página creada, fue creada para el negocio de un familiar que requería de una web para atraer clientes. El negocio era principalmente de venta de pollo, pero también ofrecía otros productos los cuales se pueden ver en la sección "Productos".

El proyecto tiene una estructura simple, todo fue creado 100% vanilla. Usé HTML, CSS aplicando metodología BEM, Mobile First y JavaScript para el manejo del DOM. Para la parte de los sliders y el filtro de la sección de productos se usó JavaScript.

Programa de escritorio para Gestor de inventario

Video de demostración:
Un poco sobre el proyecto

Este proyecto se está realizando con el fin de agilizar la gestión del inventario de un negocio de venta de productos cárnicos, especialmente pollo. Hasta ahora el negocio maneja el inventario de manera manual (en cuadernos fisicos). Con este programa automatizaría la entrada y salida de producto, así como las cuentas de las ventas y compras de los mismos.

Para el desarrollo se usó el lenguaje Java, aplicando el patron de diseño MVC o Modelo-Vista-Controlador, el diseño de la interfaz se hizo en Figma, se usó Java Swing para aplicar el diseño de figma dentro de Java. Todas las entradas, salidas y registros en general se almacenan en una base de datos de MySQL.

Principales caracteristicas o funciones del programa:
  • Agregar productos
  • Agregar clientes
  • Agregar proveedores
  • Realizar entradas de producto
  • Realizar salidas de producto
  • Generar reportes en PDF

NOTA: Actualmente está en desarrollo, haciendo mejoras y arreglando detalles.

Tecnologías usadas
Java
MySQL
MVC(Modelo-Vista-Controlador)
Figma
